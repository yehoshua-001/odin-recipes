# Project: Recipes | The Odin Project
In this project I will make a static website about "Dish/cuisine Recipes" using HTML and later on CSS with different layout properties to apply, demonstrate, and practice what I learned in [The Odin Project: Foundations Course (HTML and CSS modules) ](https://www.theodinproject.com/paths/foundations/courses/foundations)

## Conclusion
I applied and demonstrated the lessons I learned in HTML and CSS Foundations modules including proper file path and directories, making HTML boilerplate, using HTML elements, adding necessary information for the recipes, add images, also add relative links and absolute links (with target attribute) for referencing. For the design I used different layout properties of CSS. I also demonstrated and practiced using Git by adding, committing with messages, and pushing the changes to my repository on GitHub.
